## Enumeração Web

Ao acessar a porta 80, encontramos uma página web padrão. Vamos enumerar diretórios:

```bash
gobuster dir -u http://10.10.X.X -w /usr/share/wordlists/dirb/common.txt
```

### Diretórios Encontrados

- `/admin` - Página de login administrativa
- `/uploads` - Diretório de uploads (listável)

## Exploração

Após análise da aplicação, identificamos uma vulnerabilidade de upload de arquivos que permite executar código no servidor.

> **Nota:** O processo de exploração foi simplificado para fins educacionais. Em um cenário real, sempre documente cada passo detalhadamente.

### Passos da Exploração

1. Identificar o tipo de arquivo permitido
2. Criar um payload adequado
3. Fazer upload e executar
4. Obter shell reverso

## Escalação de Privilégios

Uma vez com acesso ao sistema, verificamos permissões sudo:

```bash
sudo -l
```

O usuário possui permissão para executar um binário específico como root, o que nos permite escalar privilégios.
